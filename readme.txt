== Installation ==

1. Upload the `ms-gallery-for-divi-lite` folder to the `/wp-content/plugins/` directory.  
2. Activate the plugin through the **Plugins** menu in WordPress.  
3. Open the Divi Builder and add the **Ms Gallery** module to your page.  
4. Select your images and customize the settings.  
5. Save and publish your page. Enjoy your new gallery!

== Frequently Asked Questions ==

= How do I add a gallery to my Divi page? =  
After activating the plugin, open Divi Builder and add the "Ms Gallery" module. Select your images and customize the layout.

= Can I customize the look of my gallery? =  
Yes, you can set the number of columns, adjust spacing, choose image sizes, add borders, and apply hover effects.

= Is the plugin responsive? =  
Yes. The gallery adjusts automatically for desktop, tablet, and mobile screen sizes.

= What image formats are supported? =  
All standard image formats supported by WordPress: JPG, PNG, GIF, and WebP.

= How many images can I add? =  
Thereâ€™s no hard limit. For performance, we recommend keeping galleries under 50 images.

= Can I mix image sizes in the gallery? =  
You can select one WordPress image size per gallery, but images with different dimensions can be used for a masonry effect.

= Does it work with the Divi Theme and Divi Builder plugin? =  
Yes. Compatible with both the full Divi Theme and the standalone Divi Builder plugin.

= Can I change hover effects? =  
The plugin includes default hover effects. You can further customize them using Diviâ€™s Advanced CSS options.
